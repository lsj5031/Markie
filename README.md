## 🎨 Why Markie?

Writing content in Markdown is fast and efficient, but generating beautiful, print-ready or social-media-ready visuals from that Markdown can be tedious. **Markie** bridges the gap by providing a curated selection of stunning typography themes and an intuitive real-time preview, allowing you to instantly export your content into beautifully styled graphics.

Whether you're crafting an editorial piece, a technical document, or a social media carousel, Markie ensures your content looks incredibly professional with zero design effort.

## ✨ Features

- **🖋️ 14 Curated Themes**: Choose from Classic Editorial, Bauhaus Redux, Cyberpunk Neon, Retro Futurism, Dark Academia, and more.
- **⚡ Real-Time Preview**: See your Markdown rendered instantly, exactly as it will appear in the final export.
- **📄 Multiple Export Modes**: 
  - *Pages Mode*: Automatically paginates your content for seamless carousels or documents.
  - *Continuous Mode*: Exports one long image, perfect for long-form social media or blogs.
- **📐 Export Sizes**: Support for standard A4 document sizes or 1:1 Square layouts designed for Instagram/LinkedIn.
- **🖼️ High-Quality Exports**: Export crisp, high-resolution PNGs or resolution-independent SVGs with customizable padding.
- **📱 Responsive Editor**: Built with a collapsible sidebar and clean interface to keep you focused on writing.

## 🛠️ Tech Stack

Markie is built with modern web technologies:
- **Core**: [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Markdown Handling**: `marked.js` with `DOMPurify` for secure parsing
- **Exporting**: `html-to-image` optimized for custom pagination
- **Testing**: Playwright e2e tests
- **Quality**: Oxlint + Husky pre-commit hooks

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
- Node.js (v18 or higher recommended)

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/lsj5031/Markie.git
   ```
2. Navigate to the project directory
   ```sh
   cd Markie
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```
